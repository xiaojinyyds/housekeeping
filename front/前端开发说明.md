# 家政管理系统前端 - 开发说明

## 技术栈

- **框架**: Vue 3 + TypeScript
- **UI组件**: Vant 4 (移动端)
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP客户端**: Axios
- **构建工具**: Vite

## 已完成配置

### 1. API接口模块
- ✅ `src/api/modules/auth.ts` - 认证相关接口
- ✅ `src/api/modules/upload.ts` - 文件上传接口
- ✅ `src/api/modules/worker.ts` - 家政阿姨相关接口

### 2. 用户状态管理
- ✅ `src/store/modules/user.ts` - 用户Store
  - 登录/注册
  - 用户信息管理
  - 角色判断（user/worker/admin）

### 3. 环境配置
- ✅ 后端API地址: `http://localhost:8000`
- ✅ 前端端口: `9000`

## 启动步骤

### 1. 安装依赖

```bash
cd H5-project/front
pnpm install
```

### 2. 启动开发服务器

```bash
pnpm dev
```

服务将在 **http://localhost:9000** 启动

## 用户角色说明

### 1. 普通用户 (user)
- 浏览家政阿姨列表
- 预约家政服务
- 查看预约记录
- 申请成为家政阿姨

### 2. 家政阿姨 (worker)
- 管理个人档案
- 查看预约订单
- 接单/拒单
- 设置接单状态

### 3. 管理员 (admin)
- 审核家政阿姨申请
- 管理所有用户
- 管理服务项目
- 查看系统数据

## API接口使用示例

### 登录
```typescript
import apiAuth from '@/api/modules/auth'

const login = async () => {
  try {
    const res = await apiAuth.login({
      account: 'user@example.com',
      password: 'password123'
    })
    console.log('登录成功:', res.data)
  } catch (error) {
    console.error('登录失败:', error)
  }
}
```

### 上传头像
```typescript
import apiUpload from '@/api/modules/upload'

const uploadAvatar = async (file: File) => {
  try {
    const res = await apiUpload.uploadAvatar(file)
    console.log('上传成功:', res.data.url)
  } catch (error) {
    console.error('上传失败:', error)
  }
}
```

## 用户Store使用

```typescript
import { useUserStore } from '@/store/modules/user'

const userStore = useUserStore()

// 检查登录状态
if (userStore.isLogin) {
  console.log('已登录')
}

// 检查用户角色
if (userStore.isWorker) {
  console.log('是家政阿姨')
}

if (userStore.isAdmin) {
  console.log('是管理员')
}

// 获取用户信息
console.log(userStore.userInfo)
```

## 注意事项

1. 所有API请求都会自动添加 `Authorization: Bearer <token>` 头
2. 响应格式统一为 `{ code: 200, message: 'success', data: {} }`
3. 401错误会自动退出登录并跳转到登录页
4. 文件上传需要使用 FormData 格式
